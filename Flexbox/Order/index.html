ORDER

La propiedad order en CSS se utiliza en un contexto de diseño flexible (Flexbox) o de cuadrícula (CSS Grid) para controlar el orden visual de los elementos dentro de un contenedor, sin alterar su orden en el código HTML.

Order: 0; (Valor por defecto): Los elementos se presentan en el orden en que están en el código HTML
