INTRODUCCIÓN A FLEXBOX

Flexbox es un módulo de CSS que facilita la disposición de elementos dentro de un contenedor, permitiendo alinearlos y distribuirlos de manera eficiente, incluso cuando su tamaño es dinámico o desconocido. Flexbox es particularmente útil para diseñar interfaces que se adaptan a diferentes tamaños de pantalla, como en diseño web responsive.
