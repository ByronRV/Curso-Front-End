ACCESIBILIDAD WEB

"aria-label": es un atributo de accesibilidad en HTML que proporciona una etiqueta accesible para un elemento interactivo que no tiene contenido textual visible que describa su propósito. Este atributo es especialmente útil para usuarios que utilizan lectores de pantalla. Para probarlo activamos la extension Screen Reader

Para medir la accesibilidad lo hacemos presionando la tecla F12 en la página que estamos desarrollando, luego en la sección Lighthouse, allí le damos en analizar y nos muestra el porcentaje de accesibilidad y algunos consejos para mejorarla
