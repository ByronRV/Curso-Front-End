DISPLAY

la propiedad display se utiliza para definir cómo se muestra un elemento en una página web. Controla el modelo de caja y el comportamiento de un elemento en el flujo del documento. Aquí tienes un resumen de los valores más comunes que puedes asignar a display:

display: block;

Hace que el elemento se comporte como un bloque. Ocupa todo el ancho disponible, comenzando en una nueva línea. Acepta margin, padding, etc

display: inline;

Hace que el elemento se comporte como un elemento en línea. Solo ocupa el espacio necesario y no comienza en una nueva línea. El margin y el padding se rompen, no funcionan correctamente

Enlace Enlace Enlace

display: inline-block;

Combina características de block e inline. El elemento no comienza en una nueva línea, pero puedes establecerle ancho y alto.
