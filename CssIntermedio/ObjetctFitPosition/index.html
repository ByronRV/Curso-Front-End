OBJECT FIT Y OBJECT POSITION

Se usan para ajustar imagenes o videos dentro de su contenedor

Object fit:

  • fill: Este es el valor por defecto. El contenido se ajusta para llenar el contenedor, pero puede distorsionarse si las proporciones del contenido no coinciden con las del contenedor.
  • contain: El contenido se escala para encajar dentro del contenedor sin recortarse, manteniendo sus proporciones originales. Puede dejar espacio vacío en los bordes del contenedor si las proporciones no coinciden.
  • cover: El contenido se escala para cubrir todo el contenedor, manteniendo sus proporciones originales. Parte del contenido puede recortarse si sus proporciones no coinciden con las del contenedor.
  • none: El contenido no se redimensiona. Mantiene su tamaño original, lo que podría hacer que se desborde del contenedor si es más grande.
  • scale-down: Es una combinación de none y contain. Escala el contenido para que sea el más pequeño posible, pero sin distorsionarse. El contenido solo se redimensiona si es más grande que el contenedor.

Object position:

  • top: Alinea el contenido en la parte superior del contenedor.
  • bottom: Alinea el contenido en la parte inferior del contenedor.
  • left: Alinea el contenido en la parte izquierda del contenedor.
  • right: Alinea el contenido en la parte derecha del contenedor.
  • center: (Valor por defecto) Alinea el contenido en el centro del contenedor.
  • x% y%: Posiciona el contenido según los porcentajes dados. El primer valor se refiere a la posición horizontal y el segundo a la vertical.
  • xpx ypx: Posiciona el contenido usando píxeles en lugar de porcentajes.
