METODOLOGÍA BEM

BEM (Block, Element, Modifier) es una metodología de nomenclatura y estructuración de CSS que busca mejorar la mantenibilidad, escalabilidad y claridad del código. Fue desarrollada por Yandex y es ampliamente utilizada en proyectos de desarrollo web para organizar CSS de manera más efectiva.
