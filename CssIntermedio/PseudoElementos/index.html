PSEUDO-ELEMENTOS

Los pseudoelementos en CSS son palabras clave que se añaden a un selector para aplicar estilos a una parte específica del contenido de un elemento. A diferencia de las pseudoclases, que se aplican a un estado particular de un elemento, los pseudoelementos se usan para modificar partes específicas del contenido de un elemento, como insertar contenido antes o después del contenido de un elemento, o aplicar estilos a la primera letra o línea de un bloque de texto.

Para ingresar a los pseudo-elementos se hace con doble dos puntos "::"
