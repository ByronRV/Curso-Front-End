PSEUDO-CLASES

Las pseudoclases en CSS son palabras clave que se añaden a selectores para especificar un estado especial del elemento seleccionado. Permiten aplicar estilos en función de su estado, posición, interacción del usuario, y más, sin necesidad de modificar el DOM o agregar clases adicionales en el HTML.

:hover Cambia el color de un enlace cuando el cursor está sobre él.

:active Se aplica mientras un elemento está siendo activamente pulsado por el usuario.
