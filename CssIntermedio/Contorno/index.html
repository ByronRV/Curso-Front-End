CONTORNO (OUTLINE)

se utiliza para dibujar un contorno alrededor de un elemento. A diferencia del borde (border), el contorno no afecta el flujo del documento ni ocupa espacio; es decir, no desplaza otros elementos cuando se aplica. Se usa a menudo para resaltar elementos, como en el caso de la accesibilidad al navegar con el teclado.
