POSICIÓN RELATIVA Y ABSOLUTA

La propiedad position en CSS se utiliza para especificar cómo un elemento se posiciona en el documento. Hay varios valores posibles para position, cada uno con un comportamiento diferente:

position: static;

  • Es el valor por defecto.
  • Los elementos se posicionan de acuerdo al flujo normal del documento.
  • No se pueden usar las propiedades top, right, bottom o left.

position: relative;

  • el punto de origen es su posición original en el flujo del documento.
  • Puedes usar top, right, bottom, left para ajustar la posición del elemento, pero el espacio ocupado en el flujo del documento no cambia.

position: absolute;

  • El elemento se posiciona en relación al contenedor más cercano que tenga un position diferente de static. Se apila detras del elemento mas cercano
  • Si no hay un contenedor posicionado, se posiciona respecto a la ventana del navegador.
  • Se retira del flujo normal del documento, por lo que no afecta la posición de otros elementos.
  • Su punto de origen puede ser el contenedor donde se encuentra configurandolo como absolute
